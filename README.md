# :eyes: Eye Movement Exercise

## :two_hearts: Description

A pair of blinking eyes styled with CSS, using a simple JavaScript event to follow the user's cursor around the screen.

## :question: How to run

A live version can be viewed at https://helloaimee.github.io/eye-movement.

This project has no special dependencies. Save each file to a folder on your local machine for customization.
